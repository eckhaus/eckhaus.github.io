Chef language

Chef is stack-oriented esoteric programming language designed by David Morgan-Mar in 2002. Programs in Chef look like recipes, where variables are represented by ingredients and program's stacks are referred to as mixing bowls and baking dishes. Full specification of the language can be found on David Morgan-Mar's webpage or on Steffan Mueller's interpreter pages.

Usage of the interpreter

Command line arguments In trace (-t) mode, the interpreter waits for user input after executing each line of code.

  • -i <file>: input file
  • -o <file>: output file
  • -v: show detailed information about recipes
  • -t: interactive trace mode

Example:
interpret file myInput.chef and write verbose output into file output.txt

chef -i myInput.chef -o output.txt -v

Sample recipe

Input: In verbose (-v) mode, the interpreter prints out additional information about recipes such as ingredients, commands and loops.
(factorialFibonacci.chef)

Factorial.

This recipe calculates and prints factorials of first integers.

Ingredients.
10 g max
1 g cur
1 g sf

Method.
Put cur into mixing bowl.
Put cur into 1st mixing bowl.
Put max into 1st mixing bowl.
Put sf into 1st mixing bowl.
Serve with Fac.
Fold sf into mixing bowl.
Pour contents of mixing bowl into the baking dish.

Serves 1.

Fac.

Recursion step.

Ingredients.
0 g max
0 g cur
0 g sf
1 g one
0 g next
0 g condition

Method.
Fold sf into 1st mixing bowl.
Fold max into 1st mixing bowl.
Fold cur into 1st mixing bowl.
Clean the mixing bowl.
Clean 2nd mixing bowl.
Put cur into 2nd mixing bowl.
Remove max from 2nd mixing bowl.
Fold condition into 2nd mixing bowl.
Loop the condition.
  Put cur into 2nd mixing bowl.
  Add one into 2nd mixing bowl.
  Fold next into 2nd mixing bowl.
  Clean the mixing bowl.
  Put sf into mixing bowl.
  Combine next.
  Fold sf into mixing bowl.
  Put sf into mixing bowl.
  Clean 1st mixing bowl.
  Put next into 1st mixing bowl.
  Put max into 1st mixing bowl.
  Put sf into 1st mixing bowl.
  Serve with Fac.
  Refrigerate.
Do until looped.

Serves 0.
